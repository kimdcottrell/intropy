classmethod class_method()¶

method()¶

through the self parameter, instance methods can freely access attributes and methods of the same object.

instance methods come in handy when modifying an object’s state, as well as the class state via the self.__class__ attribute

static static_method()¶

this cannot modify an object’s state nor class state.

these functions are restricted in what they can access. they’re primarily used to namespace your methods.
